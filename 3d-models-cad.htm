Implementation Guide

How to Use Our Models

Revit Family Integration

Step-by-step guide for loading and using LUNVYR families in your Revit projects.

  1. 1
    Download Family Files

    Download the .rfa files for your selected fixtures from our library. Each family includes multiple size and finish configurations.

  2. 2
    Load into Project

    In Revit, go to Insert > Load Family and select the downloaded .rfa file. The family will appear in your project browser under Lighting Fixtures.

  3. 3
    Configure Parameters

    Select the fixture and modify type parameters for size, finish, and mounting details. All dimensions update automatically.

  4. 4
    Generate Schedules

    Create lighting schedules that automatically pull fixture specifications, quantities, and custom parameters for documentation.

AutoCAD Integration

Import and use LUNVYR fixtures in your CAD drawings and construction documents.

  1. 1
    Import DWG Files

    Use INSERT > Block to add fixture blocks to your current drawing. All views are organized on separate layers.

  2. 2
    Scale and Position

    Scale blocks as needed and position in your floor plans. Each block includes plan, elevation, and detail views.

  3. 3
    Layer Management

    Organize fixtures on dedicated layers for easy visibility control. Use standard AIA layer naming conventions.

Rendering and Visualization

Set up realistic materials and lighting for photorealistic renderings.

  1. 1
    Material Setup

    Apply provided material libraries for accurate brass finishes, patinas, and surface properties in your renderer.

  2. 2
    Lighting Configuration

    Configure emissive properties and IES light distribution files for realistic light output and beam patterns.

  3. 3
    Render Settings

    Use recommended render settings for optimal quality and reasonable render times with our high-detail models.
