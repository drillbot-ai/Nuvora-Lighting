    Setup Instructions

    Installation Guide

    Dimmer Installation & Setup

    Professional installation steps for optimal dimming performance with LUNVYR fixtures.

    1. 1
      Power Safety

      Turn off power at the circuit breaker and verify with a non-contact voltage tester. Safety is paramount when working with electrical systems.

    2. 2
      Wire Installation

      Connect dimmer according to manufacturer's wiring diagram. Ensure proper neutral connections for LED compatibility.

    3. 3
      Load Configuration

      Set dimmer to LED mode if available. Adjust minimum and maximum light levels for optimal performance range.

    4. 4
      Testing & Calibration

      Test full dimming range, checking for flicker, noise, or compatibility issues. Fine-tune settings as needed.

    Smart Home Integration

    Connect your LUNVYR lighting to smart home systems for voice control and automation.

    1. 1
      Hub Connection

      Connect smart dimmer to your home network or hub following manufacturer's pairing instructions.

    2. 2
      Device Discovery

      Use your smart home app to discover and add the new lighting devices to your system.

    3. 3
      Room Assignment

      Assign fixtures to appropriate rooms and create meaningful names for easy voice control.

    4. 4
      Scene Creation

      Set up lighting scenes for different activities like dining, entertaining, or relaxing.

    Testing & Optimization

    Ensure optimal performance and troubleshoot any compatibility issues.

    1. 1
      Performance Testing

      Test dimming range from 1% to 100%, checking for smooth operation without flicker or noise.

    2. 2
      Load Verification

      Verify that total connected load doesn't exceed dimmer capacity. Check power consumption at various levels.

    3. 3
      Smart Function Test

      Test all smart home functions including voice commands, app control, and automated scenes.

    4. 4
      Documentation

      Document all settings and configurations for future reference and maintenance.

    Flickering or Strobing

    Lights flicker at certain dimming levels or when adjusting brightness.

    1. Verify dimmer is rated for LED loads
    2. Check minimum load requirements
    3. Adjust dimmer's low-end trim setting
    4. Consider adding a neutral wire if missing
    5. Test with different dimmer model if needed

    Buzzing or Humming

    Audible noise coming from fixture or dimmer during operation.

    1. Check if noise is from dimmer or fixture
    2. Verify proper wire connections
    3. Try different dimmer technology (MLV vs ELV)
    4. Check for loose connections in fixture
    5. Consider upgrading to premium dimmer

    Smart Control Issues

    Smart home commands not working or inconsistent response.

    1. Check Wi-Fi or hub connectivity
    2. Verify device is properly paired
    3. Update smart home app and firmware
    4. Reset and re-pair the device
    5. Check for interference from other devices

    Overheating

    Dimmer or fixture running excessively hot during operation.

    1. Check total connected load vs dimmer rating
    2. Ensure proper ventilation around dimmer
    3. Verify wire gauge is adequate for load
    4. Consider de-rating for multi-gang installations
    5. Upgrade to higher-capacity dimmer if needed

    Limited Dimming Range

    Lights don't dim to very low levels or turn off before 0%.

    1. Adjust dimmer's low-end trim setting
    2. Check for minimum load requirements
    3. Verify LED driver compatibility
    4. Try reverse-phase (ELV) dimmer
    5. Consider adding a dummy load if necessary
    üîÑ

    Inconsistent Performance

    Lighting behavior varies between similar fixtures or installations.

    1. Verify all fixtures are same model/revision
    2. Check for consistent wiring methods
    3. Ensure equal wire run lengths where possible
    4. Test each circuit independently
    5. Document any differences for future reference
